# 🛰️ Canopy Sentinel 2.0: Satellite Deforestation Monitoring & Carbon Bank

[![Kotlin](https://img.shields.io/badge/Kotlin-1.9.0-blue.svg)](https://kotlinlang.org/)
[![Android](https://img.shields.io/badge/Android-API_24+-green.svg)](https://developer.android.com/)
[![License](https://img.shields.io/badge/License-Apache_2.0-orange.svg)](https://opensource.org/licenses/Apache-2.0)

**Canopy Sentinel** is an advanced Android-based environmental monitoring platform designed to detect deforestation using real-time satellite data. It also estimates carbon loss and introduces a **Live Carbon Bank**, helping users understand both ecological and economic impact.

---

## 🚀 Key Features

* 🌍 **Live Satellite Mapping:** Visualize global forest regions with highlighted deforestation zones.
* 💰 **Carbon Impact Analysis:** Estimate carbon loss based on deforestation data and market rates.
* 🕒 **Historical Comparison:** Compare forest data over time using archived satellite data.
* ⚡ **Real-Time Alerts:** Instant notifications for deforestation activities with severity levels.
* 📑 **Report Generation:** Generate environmental reports with charts and CO₂ impact summaries.

---

## 🛠️ Tech Stack

* **Language:** Kotlin  
* **Architecture:** MVVM  
* **Mapping:** osmdroid  
* **Charts & Analytics:** MPAndroidChart  
* **Data Handling:** Android DataStore
